This is the week that separates "I built a chatbot" from "I shipped a product."

## Key Topics Tested

- Conversation memory in stateless LLM systems (buffer, sliding window, summary, injection)
- Streamlit / Gradio / frontend architecture for AI applications
- Multi-tool orchestration — routing, context-aware follow-ups, tool scaling
- Testing non-deterministic AI systems (unit, mock, integration, eval)
- The 7-step project lifecycle (scope → architect → build → test → document → deploy → tell the story)
- Deployment strategies (Streamlit Cloud, Docker, Railway, Vercel + API)
- Observability — logging, export, audit trails, session replay
- Documentation as engineering practice — not afterthought
- "Project vs Product" — the gap most candidates can't articulate
