## Key Topics Tested

- Tokenization and its downstream effects on cost, performance, and multilingual behavior
- Temperature, top-p, and sampling strategies — the math, not just the knob
- Context windows — limits, costs, and engineering strategies
- Embeddings and vector similarity for retrieval
- Hallucination — causes, detection, and production mitigations
- Training vs inference — what engineers actually need to know
- Transformer architecture — attention, positional encoding, KV-cache
- Model selection trade-offs (cost, latency, quality)
- Prompt injection defense (the #1 AI security topic)
